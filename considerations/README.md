# The Considerations

This directory hosts the canonical statement of the book's organizing thread: the **considerations** that recur across every chapter and every case study.

## Files

- **[`catalogue.md`](catalogue.md)** — the canonical considerations text. Mirrors the book's preface §"The considerations" and is the source-of-truth for what each consideration says, where the book develops it, and where the repository's executable content reinforces it.
- **`per-consideration-resources.md`** — *(populated incrementally)* a comprehensive cross-reference table mapping each consideration to the specific notebooks, exercises, figures, and datasets in this repository that develop it.

## How to navigate the repository by consideration

The repository supports two navigation modes:

**By chapter** (book-aligned): Open `python/notebooks/chapter_NN_xxx/` and read in book order.

**By consideration** (concept-aligned): Open `catalogue.md`, find the consideration of interest, and follow the resource pointers to the notebooks/exercises/figures that develop it. This mode lets readers re-trace the book's narrative thread on a single concept across multiple chapters.

## Maintenance

The catalogue text is the canonical statement of the considerations in the current draft. The set of considerations and their statements may be revised as the manuscript matures, in coordination with the author team. The cross-reference resources in `per-consideration-resources.md` are updated whenever notebooks, exercises, or figures are added.
