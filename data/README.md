# Datasets

This directory contains the datasets used in the book's case studies and exercises.

## Subdirectories

- **`covid/`** — COVID-19 outbreak data used in Chapter 17 (Integrated Case Studies: Introduction and COVID-19)
- **`dengue/`** — Dengue outbreak data used in Chapter 18
- **`hiv/`** — HIV surveillance data used in Chapter 19
- **`synthetic/`** — Programmatically-generated outbreak data with fully documented parameters and seeds, for exercises and pedagogical demonstrations

## Per-dataset structure

Each dataset folder contains:
- The data file(s) (CSV, NetCDF, or other appropriate format)
- `metadata.yaml` — data dictionary, units, time range, geographic scope
- `source_attribution.md` — citation, license, retrieval date, retrieval URL

## Licensing

- **Synthetic datasets** (`synthetic/`) are licensed under [CC BY-NC 4.0](../LICENSE-CONTENT)
- **External datasets** (`covid/`, `dengue/`, `hiv/`) retain their original source licenses, which may differ from CC BY-NC 4.0. See each `source_attribution.md` for the applicable terms.

## Population status

As of v0.6.0 (S7), `covid/`, `dengue/`, and `hiv/` contain synthetic exemplar datasets generated by `_generate_synthetic.py` from documented forward models with seeds 20260507/08/09. Each dataset includes a CSV file, a `metadata.json` documenting the generating parameters (used by the case-study notebooks for verification), and a `source_attribution.md`. The `synthetic/` subdirectory is reserved for additional pedagogical datasets and is not yet populated.
